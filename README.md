# 65472论证器
一个将任意数字分解成 65472 构成的公式的工具。

## 原项目首页
[https://lab.magiconch.com/homo/](https://lab.magiconch.com/homo/)

## 引用方式
```HTML
<script src="homo.js"></script>
<script>
let 猪肉刺身 = homo(114514);
// "65472+(654*72+(65*4*7+2+(6+54+72)))"
</script>
```
